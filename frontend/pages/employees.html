Employee Access Rules

1. Only approved employees can log in

If status is APPROVED, the employee can log in and use the portal. PENDING / REJECTED employees should not get normal access.

2. Companies are assigned employee-wise

Each employee should only see the companies assigned to them in the data submission page.

3. Admin can make another admin

If role is ADMIN, that user gets admin-side access like employee management, review, and reports.

4. Password can be changed later

While editing, leave password blank if you do not want to change the employee password.
